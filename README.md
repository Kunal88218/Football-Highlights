# ⚽ Football Statistics Dashboard

## Project Overview
The Football Statistics Dashboard is a web-based application that displays football data such as league standings, team details, and match information using a public API.

This project demonstrates core web development skills including JavaScript, API integration, and responsive UI design. Users will be able to explore football data interactively through searching, filtering, and sorting features.

---

## Purpose
The objective of this project is to:
- Work with real-world APIs using the Fetch API
- Implement JavaScript array higher-order functions such as `filter()`, `map()`, and `sort()`
- Build a clean and responsive user interface
- Create an interactive data-driven web application

---

## API Used
This project uses the **Football-Data.org API**:

https://www.football-data.org/

### API Features:
- League standings (points table)
- Team and competition details
- Match data and results
- Player-related data (limited)

---

## Project Update
Initially, the project was planned as a football highlights application using a video-based API. However, due to restricted access and API limitations, the project has been updated to use the Football-Data.org API.

The application now focuses on football statistics such as standings, teams, and matches instead of video highlights. This change ensures reliability, better data handling, and successful completion within the project timeline.
