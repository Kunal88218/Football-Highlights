# Api-Project
# ⚽ Football Highlights Web App

##  Project Overview
The Football Highlights Web App is a web-based application that displays the latest football match highlights, including goals and key moments. It uses a public API to fetch real-time data and presents it in an interactive and visually appealing interface.

This project is built to demonstrate core web development skills, including JavaScript, API integration, and UI design.

---

##  Purpose
The objective of this project is to:
- Work with real-world APIs using the Fetch API
- Apply JavaScript array higher-order functions like `filter()`, `map()`, and `sort()`
- Build a responsive and user-friendly interface
- Create an engaging application using sports data

---

##  API Used
This project uses the **Scorebat Video API**:

🔗 https://www.scorebat.com/video-api/

### API Features:
- Football match highlights
- Competition and league details
- Match titles and dates
- Embedded video clips

---

##  Planned Features

###  Search

### Filter

###  Sort

  - Newest first
  - Oldest first

###  Video Highlights
- Display embedded video highlights for each match
